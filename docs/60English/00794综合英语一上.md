+ 1、英语的名词就其**词义**来说，分为2类:
  + 普通名词

    + 前后什么都不加
    + 前面加计量词或the， **不可数名词没有复数形式，但计量词有复数形式**，two glasses of milk

### Four、This life

#### Verb Pattern4：主语+不及物动词

+ 主语+不及物动词（**不能用于被动结构**）
+ The old man can't read and write.

#### Vocabulary Exercises：-ish，re-

+ 名词 + -ish（某国家、民族或语言的；某国家的人民或语言）->名词/形容词
  + （表示一个国家的人民时，前面须加the）如，the Spanish 西班牙人
+ re-（再，重新；回复）+动词/名词/形容词->动词/名词/形容词

#### Phonetics语音

+ 6个后元音
+ 10个摩擦音
+ 音节：**说话时最小的语音片段**，通常由一个元音加上一个或几个辅音构成。
  + 单独一个元音也能成为一个音节
  + 有些较响亮的辅音，在后面没有元音的情况下也能和它前面的辅音构成一个音节
+ 单词重音：在双音节词和多音节词中，有一个音节读得特别重而强

#### Grammar：Articles冠词

+ 1、不定冠词的用法
  + 有a和an两种形式，a用于以辅音开头的单词前面，an用于以元音读音开头的单词前
  + 不定冠词与单数可数名词连用表示某一类人或事物，或泛指某人的职业
  + 与单数可数名词连用表示某一类人或事物中的一个
  + 表示数量“一”
+ 2、定冠词the的主要用法
  + 用在表示独一无二的东西的名词前
  + 用于表示海洋、河流、山脉、群岛等专有名词以及由普通名词组成的国名前
  + 用在第2次提到而变为特定的东西的名词前
  + 用在由短语或从名修饰使之成为特定的东西的名词前
  + 用在形容词、副词最高级和序数词以及only前
  + 在表示演奏乐器时，乐器的名词前用the

#### Grammar：Indefinite Pronouns不定代词

+ 0
  + **不定代词**：指的是替代或具体限定任何特定名词的代词。

+ 1、some,any及其派生词
  + any及其派生词用于肯定句时，起强调作用，意思是“任何的......”
+ 2、many和much
  + many用于可数名词前，much用于不可数名词前，**两个词主要用于否定句和疑问句**，肯定句中，用**a lot of**代替
+ 3、each和every及其派生词
  + 都表示“每个”，each强调的是作为个体来考虑的人或物，它既可作代词又可作形容词；every只能作形容词，用它表示“每个”时，强调的是全体

#### Grammar：一般现在时和现在进行时

##### The Simple Present Tense一般现在时【do/does】

+ 1、主要用来表示习惯性动作或自古已然的事态
+ 2、表示目前存在的状态
+ 3、当询问或引用书籍、通知或新近收到的信件的内容时，用一般现在时
+ 4、表示计划中将来的行动或一系列行动，尤指旅途中的行动
+ 5、常用的时间状语有always, often, every day, sometimes, usually, seldom, three times a day等

##### The Present Continuous Tense现在进行时【is doing】

+ 1、表示动作现在正在进行或发生
+ 2、表示现阶段正在进行而此刻不一定正在进行的动作，常常伴有now, these days等时间壮语
+ 3、与always, forever等表示频繁的副词连用，强调动作之经常，往往还表示说话人对此不以为然或厌烦的态度
