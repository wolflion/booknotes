#### Grammar：Nouns名词

+ 1、英语的名词就其**词义**来说，分为2类:
  + 普通名词
    + 个体名词
    + 物质名词
    + 抽象名词
    + 集体名词
  + 专有名词
+ 2、名词就**词汇形式**来说，分为：可数名词、不可数名词
  + 可数名词的用法：前面有a或the，后面带复数-s或-es
  + 不可数名词的用法:
    + 前后什么都不加
    + 前面加计量词或the， **不可数名词没有复数形式，但计量词有复数形式**，two glasses of milk
  + 集体名词的用法
