7.2、插入排序¶

7.2.1、算法¶
  • 个人思考，其实是两层
    • 外层 是数组大小（for到a.size()）
    • 内层 是把第i轮前面的都要保证有序
  • 以34，8，64，51，32，21取例，看外层的话，需要移动位置数是多少，代码实现上与C差不多
template<typename Compareble>
void insertionSort(vector<Comparable> &a){
    for(int p=1;p<a.size();++p){
        Compareable tmp = std::move(a[p]);
        int j;
        for(j=p;j>0&&tmp<a[j-1];--j){
            a[j]=std::move(a[j-1]);
        }
        a[j]=std::move(tmp);
    }
}
