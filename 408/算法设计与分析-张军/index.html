3.2.1、选择排序¶
  • 思想：从第一个元素开始扫描，找到最小的，与第一个位置的元素交换
  • 过程：84，53，68，96，32，38，16进行选择排序
    • 第一趟：找到最小的16，与第一个位置交换，即16，53，68，96，32，38，84
    • 第二躺：在剩下的里面，找到最小的32，与剩下的第一个位置交换，16，32，68，96，53，38，84
    • 以此类推
  • 伪代码，并没有完全理解，lionel
SELECTION-SORT(A)
    for i<-0 to n-2   //不太懂的是这个，为啥只到n-2次，lionel

                    then min <- j

  • 伪代码
DIJKSTRA(n,v,V,weight){
S<-{v}  //v是顶点集，S是结果集
for i <- 1 to n
    do D[i] <- weight[v][i]  //D是各顶点
        P[i] <- v
for i <- 1 to n-1
    do FIND(V-S,j)
        if D[j] =无穷大 then break
        S <- S 并 {j}
        for all k 属于 V-S
            do if D[k]>D[j]+weight[j][k]
                then D[k]<-D[j]+weight[j][k]
                    P[k] <- j
}

    1 i<-w  //lionel，不懂为啥要赋值给i  【看完懂了，主要是每次要减去一些放入的，然后才是剩下的，第一次是 最大的】

6.4.2、最长公共子序列问题¶
  • 最长公共子序列（Longest Common Subsequence）LCS
  • 我的思路：
    • 两层for循环，二维数组L[i,j]分别代表a串，b串的长度组成的数组，初始化为0
    • 如果相等就好说，L[i,j]的值=L[i-1,j-1]的值+1，即可
    • 要是不相等，就取左边或上面的那个最大值L[i,j] = max(L[i-1,j],L[i,j-1])，当时看了没懂啥意思，回来总结时懂了
  • 伪代码
LCS(L,a,b,n,m)
    for i <- 0 to n
        do L[i,0] <- 0
    for j <- 0 to m
        do L[0,j] <- 0   //为什么不直接2层for循环给 n*m 的

    //这时开始2层for循环了
    for i <- 1 to m 
        do for j <- 1 to n
            do if a[i] = b[j]
                then L[i,j] = L[i-1,j-1] + 1
               else
                L[i,j] = max(L[i-1,j],L[i,j-1]) 
    return L[n,m] //lionel，这个return到底在哪？   其实无所谓，返回的是数组里面的最后一个值
