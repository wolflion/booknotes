#### 9.2、平面图

+ 定义9.4、若一个图G能画在平面S上，且使G的边仅在端点处相交，则称图G为可嵌入平面S，**G称为可平面图**（平面图）
+ 定义9.5、**该面的边界**，包围每个面的所有边组成的回路
  + 定理9.7、设连通平面图G，面的次数之和等于其边数的两倍。
  + 定理9.8、设有一个连通平面图G，共有n个顶点和m条边，其平面表示中共有r个面，则n-m+r=2成立。此公式称为**欧拉公式**
