+ **自下而上**选股
+ 看企业研究的书

  + 《公司的核心竞争力》
  + 《蓝海战略》
+ 四个“走”，**走出去，走进去、走下去、走上去**
  + 走出去，你企业的产品市场需不需要，有没有空间
  + 走进去，你的竞争优势在哪里
  + 走下去，有没有核心能力
  + 走上去，就是蓝海，没人跟你竞争
  + **2007-2008形成的**
